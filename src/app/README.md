Project Overview

Async Race is a single-page application (SPA) for managing a collection of cars, operating their engines, and showcasing race statistics.
It allows users to:

Create, edit, and delete cars with name and color.

Start and stop car engines individually or as a race.

Track winners and race statistics.

Navigate between Garage and Winners views with preserved input states.

The UI is fully responsive and animated, supporting screens as small as 500px.

Notes on Task Timeline
